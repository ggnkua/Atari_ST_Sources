11.2.3 Mighty&Nemesis


MightySonic und die Nemesis-Karte im Falcon

Ein Artikel von
Michael Grove



For 40MHz CPU and 20MHz bus (VGA only)

1).
Connections are normal for Mighty Sonic and Nemesis (like instructions) except for High (Black) CPU clock wire for Mighty Sonic. I used the DSP clock wire from the NEMESIS for the Mighty High CPU clock (32MHZ.PRG). I habe 50MHz oscillator on DPS before, so I do not need the Nemesis DSP clock for DSP.

2).
FPU have problems mit Nemesis clock. This is normal for Nemesis (manual says). He is my fix. It may not work for others. I habe 40MHz 68882 installed. R221 must be removed. Do not connect Nemesis FPU wire. Isolate (shield) FPU clock-in pin from socket. Like the DSP, I install FPU mit own clock oscillator. Connect oscillator to 68882 clock-in pin. I use 64MHz. I tried 40MHz, 50Mhz and they did not work. This was the same without Nemesis. With Mighty at 32MHz, FPU worked best at 64MHz. ?????? This takes some experiments, but 64MHz worked for me.

3).
Clock software switching is important. Maybe I forget. This is what I remember. The Mighty must be getting 32MHz clock from Nemesis before switching to 40MHz (nemlow.prg). When Nemesis is off, normal is 32MHz for DSP (Remember I use DSP for CPU clock). So, 32MHz.PRG (for Mighty) must be run before switching Nemesis to 40MHz (nemlow.prg).

4).
Buffer mod. The Nemesis has a very nice buffer mod. The Mighty works very well with it. There are two modifications that the Nemesis manual says to make that do not work with the Mighty.

One, you place a capacitor across ground and DMA clock-in with
DMA clock wire from buffer. This is for Nemesis only.

Two, you use a resistor (no capacitor) across DMA clock-in and
ground. This is for Afterburner.

These do not work with the Mighty. Only use the DMA buffer
wire to DMA clock in. This works well on the Mighty.

5).
Autofolder
FAFAXX.PRG
NVDI.PRG
32MHz.PRG
Nemesis.PRG
Nemlow.PRG

6).
The Mighty Sonic simms and chips get VERY hot at 40MHz. A fan should be used to keep cool.

The Mighty can run 24/24 CPU/BUS for better TC Video modes (VGA). Run Nemesis at 48MHz, but do not run 32MHz.PRG on the Mighty. Mighty gets 24MHz clock from expansion bus. My FPU will not work in this mode.

I tried to run the Mighty at 48MHz. It would work 1 of 20 times. Not reliable. I habe 50Mhz 68030, but I think the problem is the simm speed/chips on the Mighty. The Mighty runs the whole card at CPU clock speed. Maybe Nemesis mit 42,44,46 MHz clock, I don't know. Aixit Falcon and TT fast ram cards habe same problem. Will not run much over 40MHz.

Special note on Nemesis. Both oscillators must be installed even if you only use one. Nemesis will not work properly if only one is installed.

7).
RGB video. The Nemesis needs to have additional modifications for RGB video to work with 20/24 MHz bus. There are instructions in the Nemesis manual. I tried this, but it did not work for me. Maybe I make a mistake. I don't know..... The Nemesis takes a lot of soldering and cutting:(. It is like the Speed Resolution Card to install.
