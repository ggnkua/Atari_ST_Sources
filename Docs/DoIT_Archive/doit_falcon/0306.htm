3.6 Failure Checklist


Failure / Symtom Checklist Falcon F30



Eine Zusammenstellung bereits bekannter Systemfehler des Falcon, dessen Subsysteme und eine mögliche Behebung oder zumindest erfolgt eine Beschreibung der Testmöglichkeiten.
Die Kürze der einzelnen Beschreibungen ergibt sich wegen einer schnellen Übersicht. Details finden Sie jeweils in den entsprechenden Artikeln die Sie im Archiv vorfinden.

Dunkler Bildschirm
Keine Versorgungsspannung. Check PSU, (Check LED), defekter COMBO Chip
Defekter VIDEL IC (Videochip)
Genaue Fehlerquelle nur mit Testmodul auffinbar.

 
Weißer Bildschirm
VIDEL Chip, COMBO Chip, SDMA IC oder MC 68030 (CPU).
Test nur mit Testmodul und einem zweiten Rechner über RS232 möglich

 
Punkte auf dem Bildschirm
Defekte 4/14(16)MB Speicherkarte, COMBO Chip, VIDEL Chip.
Test mit Testmodul und/oder Ram-Testprogramm möglich.
Speicherkarte gegen eine Karte aus einem anderen Falcon testweise tauschen.
Siehe Memoryport.

 
Vertikale Striche auf dem Bildschirm
Defekte Speicherkarte, COMBO Chip, VIDEL Chip.
Test ausschliesslich mit Testmodul oder Ram-Testprogramm möglich.
Ramkarte testweise gegen einen andere Ramkarte austauschen.
Mögliche Fehlerquellen, Stiftleisten, schlecht gelöteter COMBO Chip.
Siehe Memoryport.

 
Eine Farbe fehlt
Testen Sie ihr Videokabel und die Steckerverbinder (Monitorstecker) zuerst.
SGC DAC, VIDEL IC. Signaltest nur mit Oszilloskop korrekt möglich.
Siehe VGA-Fehlerquellen.

 
Monochrome Monitor
Überprüfen ob bei eingesteckten Monochromen Monitor auch
der Detect Pin gegen GND gelegt wird. MFP GPIT Bit 7
Ggf. den MFP Chip auswechseln.
Siehe VGA-Fehlerquellen.

 
Wilder Bildschirm (nur Color)
COMBO Chip, Test nur mit dem Testmodul möglich. Siehe VGA-Fehlerquellen.

 
TV-Modulator
Modulator defekt. Signal fehlt. Phase locked loop.
Signale testen mit Oszilloskop.
Siehe TV-Modulator.

 
Floppy Disk

 
Bootet nicht:
Stromversorgung. AJAX IC, SDMA, PSG IC, die entsprechenden Ports der genannten ICs auf Funktion testen.
IRQ Leitungen gegen GND Kurzgeschlossen.
Geht die LED beim Zugriff an, PSG IC, Output überprüfen.
Auf Motorgeräusche achten.
Gegebenenfalls das Laufwerk austauschen.

 
Formatiert nicht:
AJAX IC, SDMA Controller, Floppy tauschen.

 
Crash nach laden:
Daten auf der Diskette defekt. AJAX IC, SDMA
oder (COMBEL) COMBO IC. Drive auswechseln.

 
Tastatur
Testmodul zeigt einen roten Bildschirm:
Defektes Keyboard. 6850 IC, MFP wechseln.
Datenübertragung ACIA <> Keyboardprozessor vorher testen.

 
Midi Schnittstelle
Roter Bildschirm:
Defekte OPTO-Koppler, 6850 IC und den Inverter 74LS06 testen/tauschen.

 
RS232 Probleme
Roter Bildschirm:

 
SCC IC, 1489 Empfänger,1488 Treiber oder PSG IC.
+-12V DC Inverter defekt. (Meistens!)

 
Druckerschnittstelle
Roter Bildschirm: Defekter PSG oder MFP Chip
Schlechter Druck: Defekter ACSI Port, Kabel.
Terminierung des Ports.
Defekter 74LS244, 74LS255 Bustreiber

 
SCSI Hard Disk
Defekter SDMA Controller. COMBO Chip, AJAX IC.
5380 SCSI Controller.

 
IDE Hard Disk
Defekte CPU. COMBO Chip. Zumeist jedoch Ports am COMBO Chip zerstört.
Irreparabel, Chip muß ersetzt werden.

 
DMA Sound
+9V Spannungsregler defekt.
+- Stromversorgung (PSU) im Rechner testen.
DSP IC, CODEC IC, SDMA IC

 

Die Angabe "roter Bildschirm" bezieht sich dabei auf eine Anzeige des Atari Testmodul für den Atari Falcon F030 in der aktuellen Version. Eine passende Beschreibung zum Testkit finden sie hier ebenfalls im Archiv.

Gibt das Testkit am Schluß eines oder mehrerer Tests einen roten Bildschirm aus deutet das auf schwere Probleme oder auf einen Totalausfall des getesteten Bereichs an. Generell ist dasTestkit nicht in der Lage Systemfehler zu beheben.
