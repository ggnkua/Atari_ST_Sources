Abbildung 1 - NetWork



That all will be difficult. Don't know if you'll find an ethernet card for your Atari. Does it have a VME bus? If so you can buy a Riebl Card. Thought Best Electronics still had some. There are STing drivers available for this card. Another solution is connecting your Atari to a PC via a parallel or serial cable and using Ghostlink to do the transfert. Also thought that it should be possible with STinG. I also read on maus.sys.atari.hardware that next company developped an Atari ethernet interface. Haven't checked it out yet, maybe you can contact them once.
