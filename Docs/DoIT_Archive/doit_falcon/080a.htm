PAL / NTSC Board



I found differend parts on a PAL and a NTSC board: 

Part    Description
-----------------------------
R108    120 Ohm
R107    1.6 KOhm
R104    0 Ohm / Remove for PAL ( Insert = NTSC? )
R105    0 Ohm / Remove for PAL ( Insert = NTSC? )

L61     smal transformer
C93     68pF

I found all parts are in near of U18 / MC1377, but i can't find the real size of the parts to chance a PAL board into NTSC. I think all Falcon boards mainly the same, the only different is in the video clock circuit.

Also have a look to C233 (0.1uF) between point 14 and 7 of U65.

I think that's are all part how makes the difference between a PAL or a NTSC board.
