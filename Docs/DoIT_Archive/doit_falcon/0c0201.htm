12.2.1 Keyboard Adaptor


Falcon Keyboard Adaptors


Adaptors to use the Falcon- or a MegaST-Keyboard as external Keyboard with a Tower- or Desktop case.




All connections have been tried, and works fine, but we don't take any responsibiliy for eventually damaged equipment when using this scheme. You're doing this on your own risk!

Do not use any Keyboard cable longer as 2 meter!

NOTE:
That pictures do not show the connecting for PC-Keyboards !!!
MegaST-Keyboards don't need the Reset-Signal



Keyboard-Kabeladapter zur Verwendung der orignal Falcon Tastatur, einer MegaST-Tastatur an einem Tower- oder Desktop-Gehäuse. Diese Bilder stellen eine Verbindung von der Keyboard-Aschlußleiste des Falcon Mainboards zu einer sich im Gehäuse befindlichen Anschlußbuchse für eine externe Tastatur her. Das Anschlußkabel ab dieser Buchse zur eigentlichen Tastatur kann analog dazu angefertigt werden. Verwenden Sie dazu eine Handelsübliche DIN-Buchse. Zum Anschluß eines MegaST-Keyoards bietet sich eine Western-Anschlußbuchse an. Beides erhält man im einschlägigen Fachhandel.

Es kann keine Garantie für zerschossende Keyboards und Rechner übernommen werden. Alle Verbindungen sind mehrfach getestet. Sie benutzen diese Anleitung auf eigene Verantwortung.

ACHTUNG:
Diese Bilder zeigen nicht den Anschluß eines PC-Keyboards !!!
Benutzen Sie keine Tastaturkabel die länger als 2 Meter sind!
MegaST-Keyboards benötigen kein Reset-Signal!
Bei einer MegaST-Tastatur am Falcon kann die FDD/HDD-LED gesondert verschaltet werden
