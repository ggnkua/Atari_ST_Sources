12.2.2 PC Keyboard Adaptor


PC Keyboard Interface für Atari Computer



Keyboard Interface

Pictures are below!

With the Keyboard Interface you can connect any PC AT keyboard to your Atari STFM, STE, Mega ST, Mega STE, TT, Falcon or compatible computer. The unit supports the standard Atari mouse or any MS or PC serial mouse. A joystick connector is also provided to accept a standard Atari joystick.

List of features

Enables almost any PC AT keyboard, as used on most IBM PC compatibles, to be plugged in directly. No special driver software required.
Easy plug in installation for all computer models.
Special mode allows operation of mouse or joystick using the numeric keypad, even if no mouse or joystick is actually connected. Fixes Atari keyboard bugs such as the annoying reverse mouse movement when moving the mouse quickly.

Connectors on the interface

Standard AT keyboard connector.
Standard Atari mouse or joystick connector.
Serial mouse connector.
Computer interface connector (goes to Atari computer).

PC Tastatur Interface

Bilder folgen weiter unten

Mit diesem Tastaturinterface kann jede PC AT-Tastatur an ihrem Atari STFM, STE, Mega ST, Mega STE, TT, Falcon oder einem kompatiblen Computer angeschlossen werden. Die Einheit unterstützt die standard Atari Maus oder jede ander MS oder PC Maus. Ebenfalls ist ein Joystickanschluß vorhanden an dem jeder standard Atari Joystick betrieben werden kann.

Liste der Besonderheiten

Es kann jedes PC AT Keyboard, zumeist sind das IBM PC kompatible, sofort und direkt an diesem Interface angeschlossenwerden. Es ist keine Software oder ein Treiber erforderlich!
Einfache Plug & Play installation an jedem Atari Computer.
Ein spezieller Modus erlaubt es den Ziffenblock als Maus-Ersatz zu benutzen, daß ist immer dann möglich wenn kein Joystick oder eine Maus angeschlossen ist.

Behebt den Tastaturfehler welcher bei schnellen Mausbewegungen den Mauszeiger auf dem Bildschirm rückwärs laufen lässt.

Anschlußstecker an diesem Interface

Standard AT Tastaturanschluß.
Standard Atari Maus- und Joystickanschluß.
Serieller Mausanschluß.
Computer Interfaceanschluß (Die Verbindung zum Atari Computer).

Kombiniertes Interface

Das PC AT Tastatur- und serielle Mausinterface von Mario Becroft trägt seit einigen Tagen den Namen "QWERTYX". Nach einer längeren Entwicklungsphase entstand hier ein ultimatives Interface das an allen Atari STFM, STE, Mega ST, Mega STE, TT, Falcon und kompatiblen Atari Computern eingesetzt werden kann.

Technische Merkmale:

Anschließbar an STFM, STe or Falcon über den interen Tastaturanschluß. An Mega ST, Mega STe und TT Computern erfolgt der Anschluß über den dort schon extern vorhandenen Tastaturanschluß.

Die Installation erfolgt an allen Computer durch einfaches einstecken.

* Das Interface eignet sich für die gängigen PC AT Keyboards.
* PS/2 Keyboards können über einen Adapter angesteckt werden.

* Das Interface eignet sich für die meisten MS oder PC seriellen Mäuse. Mäuse mit PS/2 Stecker benötigen einen Adapterstecker.

* Mäuse mit 3 Tasten werden unterstützt.

* Standard Atari Mäuse und Joysticks können weiter verwendet werden.

* Konfigurbare Tastenbelegung der Tastatur. Spezifische Länderbelegungen oder rein Atari spezifische Belegungen können einprogrammiert werden. Eine Ebenumschaltung über Shift, Control oder Alternate Taste ist Eingebaut.

* Maus und Joystick können mit den Pfeiltasten einer PC Tastatur emuliert werden. Der Nummernblock ist ebenfalls voll nutzbar und kann auch direkt zum Steuern des Cursors benutzt werden.

* Alle Tastatureinstellungen können leicht mit einem GEM Konfigurationsprogramm vorgenommen werden.
