7.2.3 Yamaha CDRW 6416


Yamaha CDRW 6416 Recorder


I would appreciate some informed advice to help with troubleshooting a brand new CD-RW drive (Yamaha CRW6416SX).

But as soon as I connected the Yamaha I started getting new problems. The computer would freeze while the drive was connected and turned on; it would freeze while beginning a burn; it would freeze in CD-Recorder2 before beginning a burn; start-up problems (freezes before starting to load extensions.

Eventually I realized that most of the freezes go away if I just turn off the drive: miraculously the computer comes back to life! It also behaves better when no other SCSI peripherals are connected. This all sounds like some kind of SCSI problem.

This drive has manual termination and an easy back-panel device to select the SCSI ID.


Problem Fix:

Yamaha 6416s came with driver version 1.0b
This release contain some SCSI Bugs.

Use HDDriver SCSI test to see which version driver you have!
Use ExtenDOS test to see whitch version driver you have!

Well, if your recorder have firmware 1.0b as well. Went to Yamaha web site, download and install release 1.0c. After install these problems are gone. Also you can use any free SCSI ID for your YM6416.



Connection

YM6416 is a normal SCSI device. You should connect it to your SCSI chain as describe in some other SCSI articles here in archive. No special hints. Watch SCSI ID, Termination and lenght of Bus, burner will work well.
