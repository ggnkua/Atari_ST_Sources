Herzlichen Glückwunsch!

Sie haben so eben DOITF030 erfolgreich entpackt. Kopieren Sie den DOITF030-Ordner an jede beliebige Stelle auf Ihrer Festplatte. Alle Verkettungen zwischen allen anderen DOIT-Archiven funktionieren nur wenn sich alle nötigen Hauptordner im Root ihrer Festplatten befinden.

Orientieren Sie sich hierzu bitte an der folgenden Grafik


DOITF030.HTM muß sich dazu im gleichen Ordner wie alle anderen .HTM-Dateien befinden. Benutzen Sie einen beliebigen HTML Browser. Die HTML-Seiten mit viel Grafikanteil benötigen auf normalen 16Mhz-Systemen eine etwas längere Ladezeit!. Eine gute Wahl ist ein Falcon mit CENTurbo2 Beschleuniger bei einer Auflösung von 800x600x256

Sie werden in DOITF030 alle wichtigen Informationen zur Falcon Hardware finden. Neben Kabel- und Pinbelegungen aller Schnittstellen und einer Beschreibung der Custom Chips findet man in DOITF030 auch komplette Reparaturanleitungen für die unterschiedlichen Subsysteme des Atari Falcon. Diverse Bau- und Umbauanleitungen sowie Insidertips runden hier das Angebot ab.

Sie können sich DOITF030 mit jedem Browser auf jeder anderen Hardware betrachten.
