CD-Formate

Hier finden Sie die verschiedenen CD Formate und eine kurze Erklärung dazu. Nützlich beim erstellen von eigenen CDs.

CD-DA Audio CD
Ist das Format das von normalen Audio CD-Playern gelesen wird. Diese CDs können auf modernen CD-ROMs abgespielt werden.
 
Enhanced CD, CD Plus
Ist in allen Fällen eine CD mit zwei Sessions. Die erste Session enthält die Audiodaten während die zweite Session immer im HSF+, ISO- oder einen HYBRID-Daten Track beschrieben wird. (CD-Spiele)
 
ISO 9660
Das wohl gängigste CD-Format. Dieses Format kann von Windows über Atari, UNIX und MacOS gelesen werden. ISO-Sessions können auch als separate Partitionen auf einer CD bestehen. Eignet sich sehr gut für BackUps.
 
HYBRID-CD
Eine HYBRID-CD ist immer eine CD die z.B ein HSF, ein HSF+ und einen ISO-Teil enthält. Beides wird gemeinsam in einer Session auf die CD geschrieben. Es ist so möglich das ein Mac nur Daten auf der CD sieht die für MacOS bestimmt sind wie auch ein Atari oder ein PC nur die Daten sieht die nur dafür bestimmt sind. Gleiche Dateien müssen so nur ein einziges mal auf die CD geschrieben werden.
 
HSF+
Dateisystem für MacOS. Die neue und platzsparende Version dieses Formats. HSF+ Festplatten können 1:1 auf eine CD geschrieben werden. Gut geeignet für BackUp und Premastering.
 
HSF-Multisession
HSF-Format in mehreren Sessions schreiben. Die einzelnen Sessions werden vom MacOS als einzelne Partitionen genommen und auch so auf dem Schreibtisch gemountet.
 
Track-at-Once
Track-at-once eignet sich nicht zum erstellen einer Audiomaster CD. Bei Track-at-once entsteht nach jedem geschriebenen Track eine kleine Pause von 2 Sekunden.
 
Disc-at-once
Schreibt eine ganze CD in einem Rutsch, eine Session. Eignet sich sehr gut zum erstellen von Audiomaster CDs. Die Pausen zwischen den Tracks sind zwischen 0 und 8 Sekunden einstellbar.
 
Session-at-once
In der Hauptsache für CDs im CD-Extra Format genutzt. Zuerst werden ALLE Audiodaten in einem Rutsch auf die CD geschrieben. Nach eine Pause wird der Datenteil geschrieben. Diese CDs bestehen daher auch immer aus zwei Sessions.
 
Packet-Writing
Eignet sich gut um kleine Datenpackete auf eine CD zu schreiben. Einzigst in diesem Mode ist es möglich Daten direkt vom Desktop per Drag & Drop auf eine CD zu schieben und sie direkt schreiben zu lassen. Bedenken Sie hierbei das Ihr CD-Brenner diesen Mode beherrschen mu‚, ansonsten geht das nicht!

DVD
Auf einem DVD-Medium haben Audio, Video und Daten die gleiche Priorität. Ein normales DVD-Medium hält 4.7 Gigabyte Daten. 2.3 GB pro Seite!
Zum Abspielen von DVD-Video wird eine MPEG2-Decoder-Karte benötigt!
DVD-Audio schnappt 74 Minuten Musik bei bis zu 6 Kanälen für Surround bei einer Abtastrate von 196KHz und einer Auflösung von 24Bit. (AudioCD 44KHz und 16Bit!)

DVD-R

 
Ist das Format der heute käuflichen DVD-Videofilme.
 
DVD-RAM

 
Auf einem DVD-Medium haben Audio, Video und Daten die gleiche Priorität. Ein normales DVD-Medium hält 4.7 Gigabyte Daten. 2.3 GB pro Seite!
Zum Abspielen von DVD-Video wird eine MPEG2-Decoder-Karte benötigt!
DVD-Audio schnappt 74 Minuten Musik bei bis zu 6 Kanälen für Surround bei einer Abtastrate von 196KHz und einer Auflösung von 24Bit. (AudioCD 44KHz und 16Bit!) Auf einem DVD-RAM sind die Daten löschbar. Das Medium ist beidseitig wiederbeschreibar.
