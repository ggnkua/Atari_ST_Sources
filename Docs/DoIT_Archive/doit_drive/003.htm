Master & Slave

In einem (E)IDE System, an einem Buskabel, dürfen maximal zwei Festplatten, bzw. zwei IDE- oder ATAPI- Geräte angeschlossen werden. An diesen Gerätem muss eine aktuelle Konfiguration eingestellt sein, d.h. eine Festplatte muss wissen, ob sie allein am betreffenden Kanal angeschlossen ist (Master), oder ob es ein weiteres Gerät gibt. Ist das der Fall, so mu‚ eine bestimmte Reihenfolge festgelegt werden ohne die es nicht Funktionieren wird.

Die erste Festplatte, die von der sie booten, wird zum Master ernannt uns auch per Jumper so eingestellt. Die zweite Festplatte wird zum Slave bestimmt. Auch hier mu‚ zwingend der entsprechende Jumper gesetzt werden.

Nur bei Einhaltung dieser Reihenfolge ist ein störungsfreier Betrieb möglich.

Weiterhin bieten moderne IDE-Geräte die Möglichkeit "CS" an. CS bedeutet CableSelect, alle Geräte an einem Bus müssen in diesem Mode arbeiten bzw. müssen so gejumpert werden damit dies funktionieren kann. Bei der Einstellung "CS" regeln das die Laufwerke bze. der Controller unter sich, welches Gerät sich wann angesprochen fühlen darf.

Diese Einstellung funktioniert am IDE BUS 0 (primary) gleichermassen wie am IDE BUS 1 (secondary)

Beispiele für eine Busbelegung:

Beispiel 1
Bus 0: HDD Master
Bus 0: HDD Slave
Bus 1: CDROM Single

 
Beispiel 2
Bus 0: HDD Single
Bus 1: CDROM Single

 


Viele ältere CDROMs besitzen keine "Single" Einstellung. In diesem Fall laufen sie meisten nur mit der Einstellung als "Master" korrekt. Da zumeist auch langsamer, verfrachtet man diese Geräte, zumindest bei PCs, eben gleich auf den zweiten IDE-Port, sind diese Geräte dort alleine, spielt die Einstellung des Jumpers keine grössere Rolle.



CS / CableSelect

Weiterhin bieten moderne IDE-Geräte die Möglichkeit "CS" an. CS bedeutet CableSelect, alle Geräte an einem Bus müssen in diesem Mode arbeiten bzw. müssen so gejumpert werden damit dies funktionieren kann. Sollten sie also einen moderen PC oder Macintosh finden bei dem auf den IDE-Laufwerken KEIN Jumper bzw. der CS Jumper gesteckt ist, besonders wenn mehr als ein IDE-Gerät eingebaut ist, können sie davon ausgehen das hier "CableSelect" im Spiel ist.

Der CableSelect Jumper ist dann auf IDE Geräten zu stecken die hinzugefügt werden. Wie der Jumper gesteckt werden mu‚ findet sich auf den IDE-Geräten aufgedruckt.

Ein gestecker "Master" bzw. "Slave" Jumper führt unweigerlich zu Fehlfunktionen.
