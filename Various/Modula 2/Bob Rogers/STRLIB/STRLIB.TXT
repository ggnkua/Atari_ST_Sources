StrLib : Modula-2 String Functions

Some of these procedures were written because I didn't like the 
syntax of the Strings procedures provided with TDI's Modula-2 
compilers.  Others were written to provide additional string
operations.  Complete descriptions can be found in the definition
module.  

A simple benchmark (executing each of the Strings procedures
or the equivalent StrLib procedures 2500 times) showed the code size
using StrLib to be slightly bigger (Strings = 4260 bytes, StrLib =
5196 bytes) (optimize option set on linker); execution times for the
StrLib version were slightly shorter (StrLib = 10 seconds, Strings =
12 seconds).
