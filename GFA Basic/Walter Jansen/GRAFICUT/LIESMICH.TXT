 GRAFICUT läuft nur auf Monochrom-Monitor.

 Wenn GRAFICUT.PRG gestartet wird, mu× sich GC.RSC im gleichen Ordner
 oder Verzeichnis befinden. Wenn GRAFICUT.BAS (oder GRAFICUT.LST) vom
 GFA-BASIC-Interpreter gestartet wird, mu× sich GC.RSC im gleichen
 Ordner oder Verzeichnis wie der Interpreter befinden.
 
 File-Menü
 =========

        Select drives - Auswahl des Laufwerks zum Laden des Bildes bzw.
                zum Abspeichern des Ausschnitts. Laufwerk jeweils durch
                Anklicken auswählen, OK zum Bestätigen der Auswahl an-
                klicken, Anklicken von CANCEL lä×t die bestehende Auswahl
                unverändert.

        Select formats - Auswahl des Dateiformats zum Laden des Bildes
                bzw. Abspeichern des Ausschnitts. Auswahl durch Anklicken
                der entsprechenden Felder. OK anklicken zum Bestätigen,
                Anklicken von CANCEL lä×t die bestehende Auswahl unver-
                ändert.

                Bilddatei-Formate:
                
                Screen - Bildschirmkopie, 32000 Bytes

                DEGAS - unkomprimiertes, monochromes DEGAS-Bild (*.PI3)

                Printmaster - Printmaster-Library (*.SDR)


                Ausgabedatei-Formate:

                FleetStreet - Ausschnitt kann vom FleetStreet DTP-Programm
                        mit 'Merge picture' geladen werden

                GfA Put - Ausschnitt kann mit GfA-BASIC weiterverwendet
                        werden

                BitImg (RCS) - Ausschnitt kann als Bit Image ins RCS geladen
                        werden

                Icon (RCS) - Ausschnitt kann als Icon ins RCS geladen werden.
                        Es werden zwei Dateien geschrieben, die erste ent-
                        hält die Icon-Daten. Die zweite erhält ein M an das
                        Ende des Dateinamens und enthält die Mask-Daten.

                *** Bitte beachten: Das RCS kann Bit Image/Icon-Dateien nur
                        bis zu einer gewissen Grö×e verarbeiten. Falls der
                        Ausschnitt diese Grö×e überschreitet, wird eine 
                        Warnung angezeigt.

                1stWord - Ausschnitt kann von 1stWord mit 'Read picture'
                        eingelesen werden. (Nur 1stWord V1.16!)

        Load picture - lädt ein Bild mit dem ausgewählten Format

        Save cutout - speichert einen Ausschnitt im ausgewählten Format


 Extras-Menü
 ===========

        Show cutout - zeigt den Ausschnitt (falls vorhanden) auf grauem
                Hintergrund. Zurück mit rechter Maustaste.

        Invert screen - vertauscht schwarz und wei×. Kann für manche Bilder
                nützlich sein. Dieser Menüpunkt ist inaktiv, wenn eine
                Printmaster-Library geladen ist.


 Ausschnitt aus DEGAS-Bild oder Bildschirmkopie
 ==============================================

        Wenn ein Bild geladen ist, wird die Menüleiste angezeigt. Zum 
        Ausschneiden Menüleiste mit rechter Maustaste deaktivieren.
        Ein Fadenkreuz-Cursor wird angezeigt. Cursor auf eine Ecke des
        gewünschten Ausschnitts positionieren, linke Maustaste drücken
        und Box in gewünschter Grö×e aufziehen. Wenn der gewünschte Aus-
        schnitt umrahmt ist, Maustaste loslassen, der Rahmen verschwindet.
        Ausschnitt anzeigen: Menüleiste mit rechter Maustaste aktivieren,
        unter 'Extras' 'Show cutout' anwählen.
        Ausschnitt speichern: Menüleiste mit rechter Maustaste aktivieren,
        unter 'File' 'Save cutout' anwählen.


 Ausschnitt aus Printmaster-Library
 ==================================

        Im allgemeinen wie für DEGAS/Bildschirmkopie beschrieben.
        Es können jedoch nur 36 Printmaster-Grafiken auf dem Bildschirm
        dargestellt werden. Wenn eine Printmaster-Library mehr als 36
        Grafiken enthält, sind die Pfeil-Boxen am unteren rechten Rand des
        Bildschirms aktiv. Mit diesen Pfeilen kann, bei deaktivierter
        Menüleiste, durch die Library geblätter werden.
        Ausschneiden, Speichern und Anzeigen des Ausschnitts arbeiten wie
        für DEGAS/Bildschirmkopie beschrieben.
