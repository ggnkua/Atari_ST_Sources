Lieber GFA- Basic- Freund,

das Programm "BALKEN.GFA" ist in GFA- Basic V.3.0 geschrieben
und dient dazu, Balkendiagramme zu erstellen.

Beim ersten Einstieg in das Programm wird ein Muster vorgeführt,
wie ein solches Diagramm aussehen kann. Dieser Programmteil wird
überschlagen, wenn von einer vorhandenen Datei eingelesen wird.
Dann wird sofort das früher gespeicherte Bild erzeugt.

Wird ein neues Bild erzeugt, so führt ein lineares Menü von
Eingabe zu Eingabe. Hierzu ist keine Erklärung erforderlich.
Die Balkenhöhe wird durch Bewegen der Maus bestimmt. Die genaue
Höhe kann unter dem Balken abgelesen werden. Wird die linke
Maustaste gedrückt, so wird der Balken verändert; ein Druck auf
die linke Maustaste übernimmt die aktuelle Höhe.
Leicht negative Balkenhöhen sind ebenfalls möglich.

Ist das Diagramm fertig erstellt, so erscheint eine Menüleiste,
mit deren Hilfe korrigiert, ausgedruckt, abgespeichert und das
Programm beendet werden kann. Ausgedruckt wird mit Hilfe der
HARDCOPY- Routine, die auf die im Desktop definierte 
Schnittstelle mit den dort festgelegten Parametern zugreift.
Die Druckerparameter können auch innerhalb des Korrekturmenüs
eingestellt werden. Während des Ausdrucks ist die Menüleiste
und der Mauspfeil unterdrückt. Der Ausdruck kann mit der Tasten-
kombination <ALT>+<SHIFT>+<HELP> gestoppt werden.

Dieser Text wurde mit meinem Texteditor "LINED" geschrieben.
