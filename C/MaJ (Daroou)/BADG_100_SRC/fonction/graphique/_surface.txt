Fcm_create surface (10/11/2023)
---------------------------------


Si Fcmgfx_CreateSurface_ram_type_select = SURFACE_RAM_TYPE_SELECT_AUTO,
la routine Fcm_create_surface selectionne automatiquement le type de RAM en fonction
des profils des micros. Profil a ameliorer ...
ou bien on peut forcer un type de RAM : SURFACE_RAM_TYPE_SELECT_ST_RAM par exemple (voir Fcm_create_surface.h)


ouvre un bitmap offscreen VDI (v_opnbm() ) si possible/si disponible et si surface.handle_VDIoffscreen = SURFACE_OPEN_VDI_OFFSCREEN_BITMAP
SURFACE_NO_OPEN_VDI_OFFSCREEN_BITMAP si raster VDI normal.




- Offscreen test devient incompatible et doit ouvrir le mode
  SURFACE_OFFSCREEN_MODE_VDI en local comme les autres modes (ct60, MilanOS,etc...)

Profil pour machine de base
---------------------------------
Milan  : VRAM -> TT RAM -> ST RAM
TT     : ST RAM -> TT RAM
ST     : ST RAM (a voir avec les booster, si reserver en FastRAM)
STe    : ST RAM (a voir avec les booster, si reserver en FastRAM)
MegaST : ST RAM (a voir avec les booster, si reserver en FastRAM)
MegaSTe: ST RAM (a voir avec les booster, si reserver en FastRAM)
Falcon : CT060 VRAM/SuperVDIEL -> TT RAM -> ST RAM

Machine type:
-------------------------------------------------------------------
systeme.machine_type (high)
systeme.machine_subtype  (low)

http://toshyp.atari.org/en/003007.html#Cookie_2C_20_MCH
High 	Low 	Type
0x0000 	0x0000 	Atari ST (260 ST,520 ST,1040 ST,Mega ST,...)
0x0000  0x0001  STemulator (ragnar / boing bug)
0x0000 	0x4D34 	Medusa T40 without SCSI
0x0001 	0x0000 	Atari STE (1040 STE, ST-Book)
0x0001 	0x0010 	Mega STE
0x0001 	0x0100 	Sparrow (Falcon pre-production machine)
0x0002 	0x0000 	Atari TT or Hades
0x0002 	0x4D34 	Medusa T40 with SCSI
0x0003 	0x0000 	Atari-Falcon030
0x0004 	0x0000 	Milan
0x0005 	0x0000 	ARAnyM >=v0.8.5beta

ST et MegaST : systeme.machine_type=0 et systeme.machine_subtype=0
STe          : systeme.machine_type=1 et systeme.machine_subtype=0
MegaSTE      : systeme.machine_type=1 et systeme.machine_subtype=0x10
TT           : systeme.machine_type=2 et systeme.machine_subtype=0
Falcon       : systeme.machine_type=3 (et systeme.machine_subtype=0)
FireBee      : cookie _CF_  + systeme.machine_type=3 + ???
Milan        : cookie _MIL
Hades        : cookie hade
Medusa Txx   : systeme.machine_type=0 et systeme.machine_subtype=0 (cookie hade present ???)
