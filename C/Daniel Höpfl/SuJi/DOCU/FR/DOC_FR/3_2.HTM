3.2 La listeSuJi
La liste se compose de 2 zones: tête et contenu.

Tête

Dans la tête est indiqué le titre et le classement.
Il est possible de le déplacer avec un clic gauche. Un 2ème clic change le sens. (petite flèche à droite de la colonne)

Quand on laisse appuyé clic Control, alors la suite est classée par ordre alphabétique croissant ou décroissant.

Quand on appuie sur Shift-clic, la suite est classée par la date.

Une petite aide est disponible avec le clic droit, si BubbleGEM est installé.

Contenu

Quand on clique sur un des fichiers trouvés, on le sélectionne. Avec Shift on peut sélectionner plusieurs fichiers.

Avec le bouton de la souris appuyé on peut remettre les fichiers, via le Drag & Drop, à une autre application.

Un double clic (ou un clic droit) sur un fichier ouvre ce fichier. (dans la colonne "Origine" sera ouvert le dossier où le fichier se trouve)

Alt sera exploité à l'ouverture, Control doit être exploité par l'AV-Server.

Combinaisons de touches dans la liste

Control Q fermer le programme
Control U fermer le programme

Control A tout sélectionner

Control I informations sur le fichier sélectionné
TAB informations sur le fichier sélectionné

Control C copie les fichiers sélectionnés dans le presse-papiers, quand aucun fichier est sélectionné tous les fichiers sont copiés.
