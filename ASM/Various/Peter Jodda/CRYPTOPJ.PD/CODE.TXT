Beschreibung von code.prg

Das Programm ist gedacht zum verschluesseln von Files.
Es liest das File ein und erzeugt das codierte File; das
alte File wird nicht geloescht. Die Filenamen duerfen nicht
identisch sein.
Zuerst werden die beiden Filenamen eingelesen, dann wird ein
5-stelliger Schluessel gefragt. Hierfuer duerfen die ascii-codes
hex 20 - hex 59 ( space - _ ) verwendet werden. Dann wird
das File nachdem Schluessel codiert. Der Schluessel wird nicht
gespeichert, man muss sich ihn gut merken.
Das verschluesselte File wird durch erneutes codieren mit
dem gleichen Schluessel zurueckverwandelt.
