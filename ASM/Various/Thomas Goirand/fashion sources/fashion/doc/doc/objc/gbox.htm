G_BOX, G_IBOX et G_CHAR


Les objets de type G_BOX sont généralement utilisé comme "container" d'autre objet, lorsque l'on ne désire pas utilisé un G_PIX avec une texture de fond. L'ob_spec contient un champ de bits représentant la couleur de cette boite, ainsi qu'un octet signé désignant la taille du cadre.

Bit(s)DésignationSignification
0 … 15color_wordCouleur : Voire ci-dessus
16 … 23thicknessÉpaisseur du cadre


Notez que le champ "text_color" du "color_word" est ignoré, puisqu'il n'y a pas de texte dans cet objet.

Les G_IBOX sont des G_BOX sans fond (cad : fond transparent). Lorsque celles-ci sont s‚lectionnée, les objets affichés derrière elles sont affiché en inverse vidéo.

Les G_BOXCHAR sont identiques au G_BOX, et permettent de faire apparaître un caractère dans la boite.
