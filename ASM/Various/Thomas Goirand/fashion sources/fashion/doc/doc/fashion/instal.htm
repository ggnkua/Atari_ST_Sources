INSTALLATION DE FASHION


Pour installer Fashion, rien de plus simple. Choisissez un emplacement sur votre disque dur capable d'accueillir le programme (quelques centaines de Ko suffirons si vous ne voulez pas installer les fichiers d'images fournis). Copiez-y tout le dossier Fashion.

Sur le disque source vous trouverez les fichiers suivants :
  • /FASHION
    • FASHION.PRG (le programme)
    • FASHION.RSC (la ressource du programme)
    • TOOL_BOX.RSC (la ressource contenant les objets par defaut)
    • FASHION.GIF (l'image utilisé par le G_PIX par défaut)
  • /IMAGES (quelques images pour vos G_PIX)
  • /DOC (la présente documentation au format HTML)
Il est fortement recommandé de copier la documentation au format HTML sur votre disque dur pour pouvoir la lire en même temps que vous utilisez Fashion.
