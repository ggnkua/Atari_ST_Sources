Créez un menu comme décrit lors de la création d'un dialogue (en sélectionnant le radio bouton représentant un menu dans le formulaire de création), puis ouvrez-le en double cliquant sur son icône. Deux fenêtres s'ouvre alors, l'une affichant le menu, l'autre quelques boutons permettant la manipulation de ceux-ci.
Cliquer sur un titre de menu permet de déroulé celui-ci. Si vous choisissez de cr‚er un nouveau titre, celui-ci sera placé juste après le titre actuellement sélectionné.

De la même fa‡on, les entrées de menus sont placées par défaut juste après celle en cours d'édition.

Seule deux bits de l'ob_states sont éditables lorsqu'on édite des menus sous Fashion, et c'est volontaire. En effet, ce sont les seuls qui sont nécessaire à éditer sous l'AES. Si vous avez besoin d'éditer d'autre bit dans un g_string/g_title dans un menu, faites-le savoir, mais personnellement je n'en vois pas d'int‚ret.

Pour déplacer une entrée de menu, utilisez les flèches prévues à cette effet après avoir sélectionné une entrée. Si c'est un titre de menu que vous comptez déplacer, il suffit de désélectionner l'entrée en cours d'édition (seul le titre à déplacer doit être sélectionné).

Pour accéder aux labels des entrées et titre de menus, double cliquez sur ceux-ci.

Notez que le titre de menu contenant les accessoires ne doit à aucun moment être effacé sans quoi le bureau ne pourrait afficher les entrées nécessaires aux accessoires. Il est d'usage de nommer le premier titre de votre menu par le nom de votre application, ce qui permet à l'utilisateur de savoir quelle application détient le menu.
