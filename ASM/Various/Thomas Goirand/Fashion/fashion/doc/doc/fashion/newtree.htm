NOUVEL ARBRE


Trois chemins :
  • - Soit vous allez dans le menu édition, et vous cliquez sur l'option "Nouvel arbre".
  • - Soit vous utilisez le racourci clavier qui lui est assigné (affiché dans le menu).
  • - Soit encore vous cliquez dans la surface plane blanche de la fenêtre liste d'arbre (celle qui vient de s'ouvrir).
Apparaît alors une nouvelle fenêtre vous demandant de choisir un nom pour votre arbre.
  • - Vous pouvez aussi choisir quel type d'arbre vous voulez créer en cliquant sur les radios-boutons se trouvant a la gauche du bouton "OK". Pour l'instant laissez l'option par défaut (boite de dialogue, représenté par deux petits arbres).
  • - Rentrez simplement un nom pour votre boite et tapant dans le champs prévu à cet effet, puis validez en cliquant sur "OK".
Vous venez ainsi de fabriquer votre première boite de dialogue représenté dans la fenêtre de liste d'arbre de Fashion. Vous pourrez ainsi :
  • - Ajouter autant d'arbre ou de menus que vous voulez.
  • - En effacer un ou plusieurs (sélectionnez ces dialogues ou menus en cliquant une fois dessus, puis sélectionnez l'entrée "Effacer" dans le menu "Edition").
