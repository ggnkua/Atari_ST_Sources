ÉDITION D'UNE BOITE DE DIALOGUE


Pour éditer une boite de dialogue il suffit de double cliquer dans la fenêtre de liste sur un des icônes (la fenêtre se met d'abord au premier plan si elle n'y est pas). Idem lorsqu'il s'agit d'un menu.

Par défaut, un dialogue contient un G_BOX comme boite de fond. Comme nous voulons faire une interface un peu plus belle, nous allons transformer celle-ci en G_PIX pour placer une texture en fond de boite.
  • Double cliquez donc sur le G_BOX qui est l'unique objet de cette nouvelle boite. Apparaît alors un nouveau formulaire.
  • Cliquez sur le popup qui se trouve à coté des boutons "OK" et "APPLIQUE".
  • Sélectionnez l'option G_PIX. Vous voyez alors que la boite de fond de votre dialogue se transforme en une mosa‹que contenant le logo de Fashion. C'est l'image par défaut.
  • Vous pouvez changer cette image en cliquant sur le bouton "Importer image...". Un sélecteur de fichier apparaît.
  • Choisissez alors une image reconnu par Dolmen : nous recommandons les TIF, mais vous pouvez sélectionner une JPEG, une GIF, et bien d'autre encore, les types d'images étant gérés par les modules d'import/export de Dolmen placé dans le dossier "/dolmen.sys" sur votre partition de démarrage.
  • Le fond de votre boite utilisera la nouvelle image sélectionnée dès que vous aurez cliqué sur "OK" ou sur "APPLIQUER".
Une fois un joli fond sélectionné, vous pouvez placer les boutons de votre interface.
  • Pour cela placer la fenêtre contenant les objets de base au premier plan, que l'on appel toolbox (cliquez pour cela n'importe où dans cette fenêtre).
  • Puis cliquez avec le bouton droit sur l'objet que vous désirez copier, par exemple un bouton "OK", et relâchez immédiatement le bouton de la souris.
  • Déplacé-le pour qu'il soit placé au dessus de la fenêtre du dialogue en cours d'édition et re-cliquez avec le bouton droit (le bouton gauche annule l'opération). Notez que vous pouvez déplacer les objets dans les deux sens (depuis ou vers la toolbox). Dans une version ultérieur la toolbox sera sauvé automatiquement lorsque vous quitterez.
