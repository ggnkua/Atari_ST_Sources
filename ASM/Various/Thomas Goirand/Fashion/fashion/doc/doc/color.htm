COLOR WORD


Dans de nombreux objets, on peut utiliser de la couleur. Pour cela, on remplit un mot de 16 bits pr‚vu … cet effet, désignant la couleur des différents élément de cet objet. Lorsque l'objet n'utilise pas un des champs de bits (exemple : text_color dans le cas d'un G_BOX), celui-ci est tout simplement ignoré.

Ci-contre le panneau de changement de couleur utilisé dans Fashion.
Notez qu'il est utilisé pour l'ob_spec d'un objet, aussi bien que pour la tedinfo.

Bit(s)DésignationSignification
0 à 3fill_colorCouleur de fond
4 à 6patternMotif de remplissage
7opaqueA 0, l'objet est transparent, a 1, il est opaque
8 à 11text_colorCouleur du texte
12 à 15border_colorCouleur du cadre
